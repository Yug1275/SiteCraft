# 🏗️ SiteCraft

A modern construction management platform built with the MERN stack and Supabase. SiteCraft helps contractors and engineers manage projects, tasks, materials, and documents efficiently.

---

## 🚀 Features

- 🔐 Authentication (Email & Google Login)
- 📊 Project & Task Management
- 🧱 Material & Labor Tracking
- 📁 Document Management
- 📈 Reports & JSON Viewer (Upcoming)
- 🎨 Modern UI with Glassmorphism Design
